빠른 시작 Quick start

로컬에서 미니앱을 개발 중이라면 @ait-co/devtools를 추가하세요:

If you are already developing a mini-app locally, add @ait-co/devtools:

pnpm add -D @ait-co/devtools

vite.config.ts에 플러그인을 추가하면 개발 중에 SDK import가 자동으로 mock으로 swap됩니다:

Adding the plugin to vite.config.ts swaps SDK imports for mocks at dev time:

import { defineConfig } from 'vite';
import aitDevtools from '@ait-co/devtools/unplugin';

export default defineConfig({
  plugins: [aitDevtools.vite({ panel: true })],
});
